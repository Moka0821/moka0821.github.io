Moka 个人网站——GitHub Pages 上传说明

重要：你的最终网址取决于 GitHub 用户名。
例如用户名是 Moka0821：
仓库名必须是 Moka0821.github.io
网站地址就是 https://Moka0821.github.io/

步骤：
1. 登录 https://github.com/ ，右上角点“+”，选择“New repository”。
2. Repository name 填：你的GitHub用户名.github.io
3. 选择 Public，点击“Create repository”。
4. 进入新仓库后点击“uploading an existing file”或 Add file → Upload files。
5. 只上传 index.html（不要把外层文件夹一起上传）。
6. 点击绿色“Commit changes”按钮。
7. 打开仓库 Settings → Pages。
8. Build and deployment 的 Source 选“Deploy from a branch”。
9. Branch 选 main，文件夹选 /(root)，点击 Save。
10. 等待约1–10分钟，打开 https://你的GitHub用户名.github.io/

以后修改：打开仓库里的 index.html → 铅笔图标 → 修改 → Commit changes。

隐私提醒：网站会公开显示电话号码 81501641。如不希望公开，请在 index.html 中搜索并替换或删除该号码。
